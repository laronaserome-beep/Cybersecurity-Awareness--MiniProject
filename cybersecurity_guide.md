# Cybersecurity Best Practices Guide

## Password Hygiene
- Use long, unique passphrases instead of complex passwords
- Enable two-factor authentication wherever possible
- Use a password manager to store credentials securely
- Never reuse passwords across different accounts

## Email Safety
- Verify sender email addresses before clicking links
- Don't open attachments from unknown senders
- Look for spelling errors and suspicious URLs
- When in doubt, contact the organization directly

## Safe Browsing
- Look for HTTPS in website addresses
- Keep browsers and plugins updated
- Avoid downloading software from untrusted sources
- Use ad-blockers to reduce malicious ad risks
